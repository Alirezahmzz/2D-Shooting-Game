# 2D-Shooting-Game
A fun, turn-based 2D shooting game developed in C using Simple2D.
🎯 2D Turn-Based Shooting GameAbstractThis is a fun, turn-based 2D shooting game developed in C. Two players take turns adjusting their aim to shoot moving targets on the screen. The game features dynamic target movement, custom game speeds, and a built-in scoring system where hitting your opponent's target grants points, while accidentally hitting your own deducts them. It serves as a practical demonstration of basic game loops, 2D physics, collision detection, and state management in C.
✨ FeaturesTurn-Based Multiplayer: Designed for two players taking alternating turns.Customizable Gameplay: Players can set their names, game speed, and the winning score limit before starting.Basic 2D Physics & Collision: Utilizes trigonometric functions for projectile trajectories and Euclidean distance for hit detection.Save & Load State: Pressing S saves the current game progress (player names and scores) to a text file, and L loads it back.Dynamic Targets: Targets bounce off the window borders randomly based on the configured game speed.🛠️ Technologies UsedLanguage: CGraphics & Windowing: Simple2D (built on top of SDL2)   IDE: Code::Blocks
🎮 ControlsUp Arrow: Increase shooting angle.Down Arrow: Decrease shooting angle.Spacebar: Shoot projectile.S: Save current game state.L: Load previous game state.
🚀 How to RunEnsure you have the GCC compiler and the Simple2D library (along with its SDL2 dependencies) installed and configured.  Open the final.cbp project file in Code::Blocks.  Build and run the project.Follow the console prompts to input player names, speed, and target score, then press any key to launch the graphical window.
